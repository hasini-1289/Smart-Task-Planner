# Smart Task Planner (Java)

A console-based Java application to manage tasks efficiently using
priority and deadlines. This project demonstrates strong understanding
of Data Structures, Object-Oriented Programming, and clean code design.

## Features
- Add tasks with priority and deadline
- View pending tasks sorted by priority and deadline
- Menu-driven console interface
- Clean OOP-based design

## Tech Stack
- Java
- Data Structures (Priority Queue, Sorting)
- OOP Principles

## How to Run
1. Clone the repository
2. Open in any Java IDE
3. Run the Main class

## Future Enhancements
- File-based persistence
- GUI using JavaFX
